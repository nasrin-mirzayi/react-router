A simple and clean product catalog built with react. This project demonstarate routing, nested routes, dynamic routes, and navigation in a multi page react application.

📘 Features:
# Multi-page routing with React Router V7
# Nested routes using <outlet/>
# Dynamic route ( /products:id)
# Local product data ( no API )
# Back navigation using useNavigate()
# 404 --> not found page
# clean and simple components

📸 Guide:
user can visit home + product pages.
see home detals and about. then all product list.
clicking on view details to know more about cosmetics.
